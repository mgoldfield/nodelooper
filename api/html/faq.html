Can I use Zoom or other video chat software while using loopmagic!?
We have tested it with Zoom and it works well to have both zoom and loopmagic! running at the same time. Facetime created some issues that we have not investigated yet.

Can you edit tracks once they're recorded?
There are not currently plans to allow editing. If you want to edit a track, or do any other mixing/processing that's not currently available, we recommend downloading the project and throwing it in the editor/mixer of your choice.

How long to projects last?
Projects currently last for 5 days, after which they are automatically deleted. There may be a way to keep projects longer for a storage fee in the future.

How do I use MIDI instruments with loopmagic!?
You have to route the audio created by whatever MIDI software you're using to an input of your computer. You can do this physically by connecting an audio cable from your midi output to the input loopmagic! is using, or you can do this via software, explained here.

Can I record multiple channels, e.g. stereo?
Currently loopmagic! only records mono tracks. However, you can upload stereo tracks from file via "load file"
